Other stuff...

The Freemap server hosts both Freemap and OpenTrailView. However there's also a range of extra software both hosted on, and available for download from, this site. Much of it is for historical interest only, but some of it may be useful to you and it's all open source so needs to be offered for download somewhere :-)

Online software

Source code for all online software is available via Subversion, under the address svn://free-map.org.uk/svn/freemap. Please email me on nick underscore whitelegg at yahoo dot co dot uk if you would like to contribute to the code and require an SVN account. It can also be browsed online here.

Freemap 3D

A demo 3D OSM viewer using WebGL and NASA SRTM height data. Available here.

Downloadable standalone software

A range of standalone software is available for download. As said above much of it is for historical interest only but you might find some of it interesting.

FreemapMobile

A JavaME client for Freemap, written mostly in summer 2008. Available here. Shows Freemap, OSM Mapnik, Osmarender, or out-of-copyright OS maps on your Java ME-enabled phone with GPS.
